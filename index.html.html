Tout ce que vous voulez savoir

J'ai besoin de quelles compétences pour utiliser les prompts ?
Aucune. Vous copiez-collez le prompt dans ChatGPT ou Claude, vous remplissez les champs entre crochets (votre nom, le nom du produit…), et vous appuyez sur Entrée. C'est tout.
Est-ce que ça marche avec ChatGPT et Claude ?
Oui, les prompts sont conçus pour fonctionner avec les deux outils. Si vous avez une préférence, précisez-le dans le questionnaire et j'optimise pour votre outil habituel.
Comment se passe la livraison ?
Vous recevez un lien Google Docs avec vos prompts organisés par section, prêts à l'emploi. Pour les packs Business et Pro, un guide d'utilisation est inclus dans le même document.
Et si les prompts ne correspondent pas à ce que j'attendais ?
Je révise jusqu'à ce que vous soyez satisfait — dans la limite du nombre de révisions incluses dans votre pack. En pratique, une seule révision suffit dans 90% des cas grâce au questionnaire initial.
Est-ce que je peux commander pour une entreprise avec plusieurs activités ?
Oui, le pack Pro est conçu pour ça. Précisez vos différentes activités dans le questionnaire et je crée des sections séparées pour chacune.

Prêt à arrêter de réinventer la roue ?

Vos concurrents utilisent déjà l'IA. La différence, c'est d'avoir des prompts qui vous ressemblent vraiment.
